# Rumah Sakit Pratama Riung

Website statis sederhana untuk Rumah Sakit Pratama Riung dengan tampilan modern, responsif, bersih, dan profesional.

## Isi Website

- Header dan navigasi responsif
- Hero section dengan gambar tenaga medis
- Tentang Rumah Sakit Pratama Riung
- Visi dan misi
- Layanan unggulan
- Dokter dan tenaga medis
- Fasilitas rumah sakit
- Jadwal pelayanan
- Alur pelayanan pasien
- Testimoni pasien
- Kontak dan lokasi

## Struktur File

```text
RSUD-RIUNG/
├── index.html
├── styles.css
├── script.js
└── README.md
```

## Cara Menjalankan

Buka file `index.html` langsung di browser, atau gunakan ekstensi Live Server bila memakai Visual Studio Code.
